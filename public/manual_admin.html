Admin Manual

You can generate QR code that redirects the citizen to feedback form

Now you can switch on or switch off the feedback form using " feedback phase"

You can also check the filled up feedback forms by clicking on "view feedback"

If you have any query you can click on "manual" option to read the functionality

Admin will be able to see the data in Pie-chart or Bar-graph

And you can logout using "logout" option.
